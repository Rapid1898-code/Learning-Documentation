###### Set Path Variable
Start - Systemsteuerung - System - Erweiterte Systemeinstellungen
Umgebungsvariablen - Select Path Variable - Select Path in Systemvariablen - Bearbeiten
Neu - Add eg. C:\Program Files\heroku\bin
Start - Control Panel - System - Advanced system settings
Environment Variables - Select Path in SystemVariables - Edit
New - Add eg. C:\Program Files\heroku\bin

###### Shortcut for Changing KeyBoard Layout
Shift + Alt

###### Show full path of a file
Shift + RightClick and Select Copy as Path
