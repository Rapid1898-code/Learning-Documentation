Show point coordinates in Acrobat reader (in Adobe Acrobat Pro XI)
(eg. for redacting informations in a pdf using pymupdf)
- change Units to Points (Edit - Preferences - Unites & Guides - change to Points)
- show coordinates (View - Show/Hide - Cursor Coordinates)
- create a reactangle and show the coordinates

Add edit-fields (in Adobe Acrobat Pro XI)
- right upper side "Tools" - "Forms" - "Create/Edit"
- Add New Field
Hinzufügen von Edit-Feldern (in Adobe Acrobat Pro XI)
- rechts oben "Werkzeuge" - "Formulare" - Erstellen/Bearbeiten
- rechts "Neues Feld hinzufügen"

Right align text in form fields
- Properties - Options - Alignment - change to right

Set bold text in form fields
- Properties - Options - Allow Rich Text Formatting - check options
- write text in form field - select text - press Ctrl-E and format the text - delete the text
